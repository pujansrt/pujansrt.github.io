PROJECTS *

1 BPM Engine - Backend Business Process Management engine to handle various activity nodes such as Files, Web Services, REST, Java, Database etc. File related nodes provides FileSystem as well as network transfers over FTP, SFTP and FTP over SSL protocols. File handling involves various types such as XML, XLS, Fixed Width and Delimited. Regular Expression based input queue is provided through configurations. It has ability to process large volume of data.

2 Image Processing Platform for Banking Industry - Designed and implemented Image Processing Project (image admin). A web-based front-end is designed and implemented for use as a client, as well as a comprehensive API for integration with other system. ImageAdmin is built on Spring, JSP, and JMS. ImageAdmin supports client and server side rotation, re-sequencing, document type conversion, addition on the fly, printing, zoom-in/zoom-out and classification. These operations require modern browsers with no extra plugin. Also, the MVC framework front-end permits easy customization by the end-users. This system takes 6000 jobs per minutes by Job queue for processing. There is a rule engine where various document configurations are applied if required.

3 Web based Code Editor - Designed and implemented web-based code editor that supports Java, JSP, JS, CSS, XML, HTML etc. It provides compilation and run time facility for Java and JSP files. It facilitates to generate Jar on the fly. Other features include Console for monitoring, Code snippets, Repositories for code review and revisions, Remote deployment, Formatter, Prettify, Minify, Themes, Upload existing files.

4 Lightweight Enterprise Scheduler System (LESS) - Designed and implemented LESS using POJO-based framework approach. It enables to execute single or periodic tasks with queued scheduling feature which is suited for bulk processing to perform business operations in mission critical environments (banking, telecom). These tasks may include time based events such as nightly, weekly, monthly report generation, calculations, notices, logging etc. It also includes several enterprise-level features such as clustering and JTA.
